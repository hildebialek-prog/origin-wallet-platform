## Internal linking rules

Use these anchor-text patterns consistently:

- `Origin Wallet pricing` -> `/pricing`
- `Origin Wallet Business` -> `/business`
- `Origin Wallet Personal` -> `/personal`
- `Origin Wallet Help Center` -> `/help`
- `Origin Wallet security` -> `/security`
- `API and integrations` -> `/business/api`
- `multi-currency wallet` -> `/personal/wallet`
- `send money internationally` -> `/personal/send`
- `receive international payments` -> `/business/receive`
- `global supplier payments` -> `/business/suppliers`

Avoid using `click here`, `learn more`, or repeated generic CTA anchors when a keyword anchor fits naturally.

## Content writing rules

- Put the primary keyword in the title, H1, opening paragraph, and one subheading.
- Use secondary keywords naturally in body copy, not as a list dump.
- Keep one page focused on one main intent.
- Trust pages should reinforce commercial pages through links, not compete with them.
- Do not create multiple pages targeting the exact same phrase unless the intent is clearly different.
